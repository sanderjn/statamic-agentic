Help the person set up their preferences for editing this website.

Follow the onboarding steps described under **"First time here? Let's set it up"** in
`content/AGENTS.md`: ask them the short questions there, one friendly batch at a time, and save their
answers into `content/editor-notes.md`.

If `content/editor-notes.md` already has answers, don't start over — read it, then ask only what they
want to change and update those sections.

Keep it conversational and skippable. When you're done, briefly confirm what you saved.
